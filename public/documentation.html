Movie API Documentation

Welcome to the Movie API! This API allows developers to access information about movies, including details such as title, genre, release year, and ratings. You can use this API to fetch movie data and integrate it into your applications.

As we continue building this API, more endpoints and features will be added. Check back here for updates and usage instructions.

This API provides endpoints to manage movies and users.

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns data about a single movie (description, genre, director, image URL, featured flag)
GET /genres/:name Returns data about a genre (description)
GET /directors/:name Returns data about a director (bio, birth year, death year)
POST /users Allows new users to register
PUT /users/:username Allows users to update their user info (username)
POST /users/:username/movies/:movieID Adds a movie to the user’s list of favorites (returns confirmation text)
DELETE /users/:username/movies/:movieID Removes a movie from the user’s list of favorites (returns confirmation text)
DELETE /users/:username Allows an existing user to deregister (returns confirmation text)
