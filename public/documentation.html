Movie API Documentation

Welcome to the Movie API! This API allows developers to manage movies and users, including retrieving movie info, user info, and managing usersâ€™ favorite movies.

Method URL Query Parameters Request Body Response
GET /users None None Array of user objects:
[
  {
    "Username": "John",
    "Password": "hashedpassword",
    "Email": "john@example.com",
    "Birthday": "1990-01-01T00:00:00.000Z",
    "FavoriteMovies": ["Inception", "Twilight"]
  },
  ...
]
GET /users/:Username Username - user's username None Single user object:
{
  "Username": "John",
  "Password": "hashedpassword",
  "Email": "john@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": ["Inception", "Twilight"]
}
GET /movies None None Array of all movie objects:
[
  {
    "Title": "Inception",
    "Description": "...",
    "Genre": { "Name": "Sci-Fi", "Description": "..." },
    "Director": { "Name": "Christopher Nolan", "Bio": "...", "BirthYear": 1970, "DeathYear": null },
    "ImagePath": "url"
  },
  ...
]
GET /movies/:Title Title - movie title None Single movie object (same format as above).
GET /movies/genre/:genre genre - genre name None Returns Genre object for first movie found:
{ "Name": "Sci-Fi", "Description": "Science fiction movies" }
GET /movies/director/:director director - director name None Returns Director object for first movie found:
{
  "Name": "Christopher Nolan",
  "Bio": "British-American filmmaker known for...",
  "BirthYear": 1970,
  "DeathYear": null
}
POST /users None JSON object:
{
  "Username": "John",
  "Password": "password123",
  "Email": "john@example.com",
  "Birthday": "1990-01-01"
}
 Returns created user object.
POST /users/:Username/movies/:MovieID Username - user's username, MovieID - movie ID or title None Returns updated user object with the movie added to FavoriteMovies array.
PUT /users/:Username Username - user's username JSON object (fields to update):
{
  "Username": "JohnUpdated",
  "Password": "newpassword123",
  "Email": "john.updated@example.com",
  "Birthday": "1990-05-05"
}
 Returns updated user object.
DELETE /users/:Username/movies/:movieTitle Username - user's username, movieTitle - title of the movie to remove None Returns confirmation message, e.g., "Twilight has been removed from John's favorite movies."
DELETE /users/:Username Username - user's username None Returns confirmation message, e.g., "John was deleted."
