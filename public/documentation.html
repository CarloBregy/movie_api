Movie API Documentation

Welcome to the Movie API! This API allows developers to access and manage movies and users, including details such as title, genre, director, and user favorite lists.

Request URL Query Parameters Request Body Response
GET /movies None None Array of all movie objects, e.g. [{ "Title": "Inception", "Genre": {...}, "Director": {...} }, ...]
GET /movies/:title title - movie title None Movie object that matches the title, including description, genre, director, image path, and featured flag
GET /movies/genre/:name name - genre name None Array of movies with the specified genre, e.g. [{ "Title": "Se7en", "Genre": {...}, "Director": {...} }, ...]
GET /movies/genre/:genreName/director/:directorName genreName - genre, directorName - director name None Array of movies that match both the genre and director
PUT /movies/:title title - movie title { "Description": "New description text" } Updates the description of the specified movie
PUT /movies/director/:name name - director name { "Bio": "Updated bio text" } Updates the bio for all movies directed by this director
POST /users/:username/movies/:movieID username - user identifier, movieID - movie title or ObjectId None Adds the specified movie to the user's FavoriteMovies array
DELETE /users/:username username - user identifier None Deletes the specified user and returns a confirmation message
